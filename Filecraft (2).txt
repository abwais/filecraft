FileCraft-Ideen

Deine App soll werden:

FileCraft = automatische Datei-Organisation + File-Tools + Studenten-Plattform.

Kernideen:

Dateien hochladen
Dateien automatisch nach Inhalt erkennen
Dateien automatisch sortieren
Dateien automatisch umbenennen
Tags/Kategorien erstellen
PDF/File-Formatting
PDF zusammenführen
PDF komprimieren
Bilder zu PDF
Format konvertieren
Chaotischen Desktop aufräumen
Studenten-Workspaces
Uni-Gruppen nach Uni-Name
Kursräume pro Fach
Chats pro Uni/Kurs
Dateien mit Gruppen teilen
Moodle verbinden
Moodle-Aufgaben importieren
Deadlines erkennen
Erinnerungen vor Abgabe
One-Click Moodle-Abgabe
später Auto-Abgabe mit Opt-in
Backend-Schritte in sinnvoller Reihenfolge
Phase 1: Security-Fundament
JWT fertig machen
Login/Register stabilisieren
User besitzt Workspaces
User besitzt Files indirekt über Workspace
Zugriffsschutz: niemand sieht fremde Workspaces/Files

Category Entity
Tag Entity

category
tags

APIs:
Kategorie setzen
Tags setzen
Dateien nach Kategorie filtern
Suche nach Name/Inhalt/Tags

Text aus Dateien extrahieren
Datei-Inhalt analysieren

User bestätigt Vorschläge
später automatische Regeln

PDF merge
PDF compress
image-to-PDF
